Overview

ESPER is a Chrome extension designed to prevent rapid brightness changes in online videos that could trigger epileptic seizures. It dynamically processes video frames, analyzes their perceived luminosity, and adjusts brightness frame-by-frame using CSS, ensuring changes remain within a safe threshold. Additionally, ESPER collects data on video characteristics and luminosity patterns, storing them in a database for analysis through an interactive dashboard. This enables the identification of trends that could predict videos likely to induce seizures. The project combines frontend extension development, backend data processing, and data visualization for a comprehensive solution to improve video accessibility and safety.
